Admin Guide

Editing a Module

There are two ways to edit a module.

Method one: By clicking on "Add and Edit Modules" on the home page under admin specific features, all possible modules are displayed.

This will take you to the edit modules page. In the edit modules page clicking on a module will bring up the option to edit or delete the selected module

The following picture is an example of the aftermath of clicking on a module.

Method two: The alternative way of editing a module would be to go to the module itself. This can be done by going to the home page and selecting a module

Then when on the module page, selecting "Edit Module"

Following either of these methods will take you to the edit module page for the chosen module.

From here it's possible to delete the module, change the module's title, add a new image for the module

Editing Pages

To edit a page, go to the edit module page for the module that the page is in and select the page you'd like to edit. Then click edit page.

When editing a page it's possible to change the title and also change the content of the page.

To add images to a page, simply insert a link into the "Create an image" input box.

Clicking "Create an image" will auto-generate tags. Copy the tags and link into the "content" box. When you save changes a valid image link with the <img> tags surrounding it will be displayed as an image on the page.

To add videos to a page, simply insert a link from the youtube embed option in the share section into the page text.

When editing pages it's also possible to change their position in a module. When setting page positions you must ensure that each page has a unique page position. For instance a module can't contain two pages with multiple pages in position one.

It's also important to ensure that all numbers are used in positioning. For example if there are pages in positions one and three there must be a page in position two.

Adding Pages

To add a page to a module, simply go to the edit module page that you'd like to add a page to and click "Add Page". To get to the module of your choice, follow the guide above.

When adding a page you must chose the position you'd like, chose a title for the page and enter the content. An example is below. To display an image please see the "edit page" section.

Promoting Users to Administrators

To promote another user to an administrator, go to the home screen and press "Change User Privileges"

This will take you to a list of users, on this page the users current privileges are displayed. To promote/demote a user press "Change User Privileges" next to the user who you'd like to premote/demote.

Adding and Editing Quizzes

To add a Quiz, after you create a new module click on Add Quiz

This will take you to a form were you can fill out details for the quiz.

Once you click on Submit and Add Questions, two new butons will appear for adding questions. There are two types of questions (true/false and multiple choice)

If you click on any of the new options, this will take you to a form for adding a question.

Clicking submit when you edit the quesion will save that question and will take you back to the screen for editing a quiz. To save the quiz click on submit

After saving a quiz you can edit it in the Edit module page, following the same steps
